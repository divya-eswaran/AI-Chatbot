**🤖 AI Chatbot with Flask, MongoDB & NLP**

An intelligent AI-powered chatbot web application that can answer user queries using a combination of rule-based logic, FAQ matching, and NLP models.

**🚀 Features**

💬 Real-time chatbot interface

🧠 AI-based response generation (using NLP models)

📚 FAQ-based smart answering (Semantic Search)

🔍 Sentence similarity using embeddings

🗄️ MongoDB database integration

🔐 User Authentication (Login & Register system)

🎨 Modern Dark UI design

💾 Chat history storage
